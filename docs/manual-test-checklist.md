# Ручной тест-чеклист

## До публикации

- [ ] `.env` не закоммичен.
- [ ] Реальных Google-ключей нет в коде.
- [ ] `SPREADSHEET_ID` хранится в Script Properties.
- [ ] `EVENT_CODE` хранится в Script Properties.
- [ ] Реальный Web App URL не попал в репозиторий случайно.
- [ ] Google Таблица не опубликована для гостей.

## Apps Script

- [ ] `setupSpreadsheet` создает листы `Participants`, `Predictions`, `Top Ups`, `Settings`.
- [ ] `Settings` содержит `betting_open = true`.
- [ ] `Settings` содержит `seed_amount = 1000`.
- [ ] `getState` возвращает коэффициенты.
- [ ] `getState` возвращает `counts.boy` и `counts.girl`.
- [ ] `submitPrediction` записывает прогноз.
- [ ] Неверный `eventCode` не записывает прогноз.
- [ ] Пустой `eventCode` не записывает прогноз.
- [ ] `betting_open = false` запрещает отправку прогноза.
- [ ] Ошибки не показывают stack trace.

## Сайт

- [ ] Главная страница открывается.
- [ ] На мобильной версии сначала показывается форма `Ваш прогноз`, ниже — коэффициенты и голоса.
- [ ] Коэффициенты загружаются.
- [ ] В карточках коэффициентов не показываются виртуальные суммы.
- [ ] Показывается блок количества голосов.
- [ ] Без имени отправить нельзя.
- [ ] Без фамилии отправить нельзя.
- [ ] Без кода события отправить нельзя.
- [ ] Без выбора пола отправить нельзя.
- [ ] Сумма должна быть не меньше 100.
- [ ] Успешный прогноз появляется в Google Таблице.
- [ ] После прогноза коэффициенты меняются на сайте.
- [ ] После прогноза количество голосов меняется на сайте, если Apps Script возвращает `counts`.
- [ ] Кнопка отправки блокируется во время сохранения.

## Приватность публичного сайта

- [ ] Сайт не показывает имена гостей в публичной статистике.
- [ ] Сайт не показывает фамилии гостей в публичной статистике.
- [ ] Сайт не показывает список прогнозов.
- [ ] Сайт не показывает служебные настройки.
- [ ] Отдельный `display.html` не используется.
